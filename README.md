# PED Roster PWA Host

Static PWA launch page for the PED Roster Apps Script webapp.

Use this page for iPhone/Android Add to Home Screen so the shortcut uses the PED icon instead of the Apps Script initial-letter tile. The launcher offers an Add to Home Screen button and redirects to the roster after a 3-second countdown; Skip countdown opens it immediately.

The launch page redirects to the anonymous Apps Script deployment:
